# Talleres de Desarrollo con Inteligencia Artificial

Este repositorio contiene el avance, evidencias y entregables de los talleres de desarrollo asistido por Inteligencia Artificial realizados como parte del proceso de aprendizaje y adopción de prácticas modernas de ingeniería de software.

## Objetivo

Explorar y aplicar metodologías, herramientas y flujos de trabajo basados en Inteligencia Artificial para el desarrollo de software, desde escenarios individuales hasta dinámicas colaborativas en equipo.

## Contenido

### Taller 1 – Trabajo Autoguiado Individual

En este taller se desarrolla un proyecto de manera individual utilizando OpenCode para llevar a cabo los primeros pasos de creación de un agente.

**Objetivos principales:**

* Familiarizarse con agentes de IA.
* Comprender las ventajas y limitaciones del trabajo asistido por IA.
* Registrar hallazgos, aprendizajes y buenas prácticas.

### Taller 2 – Trabajo Colaborativo en Equipo

En este taller se aplica el enfoque descrito en el libro *Non-Deterministic Spec-Driven Development* de Enrico Papalini, siguiendo un proceso estructurado basado en especificaciones.

**Objetivos principales:**

* Trabajar colaborativamente utilizando IA como parte del equipo.
* Aplicar las fases y gates propuestos por el framework.
* Gestionar requisitos, planificación mediante artefactos estructurados.

**Fases aplicadas:**

1. Constitution
2. Specify (Especificar)
3. Plan (Planificar)
4. Testing design (Diseño de pruebas)

**Gates aplicados:**

* Gate 1: Validación de claridad
* Gate 2: Validación de consistencia
